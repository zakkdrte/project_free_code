About this site.

This site was created for all people that want to learn about programming languages, data bases, marckup language, css. You will learn the fundamental concepts with a few excercises.

Programming Languages

Python

A versatile, high-level language known for readability and simplicity, used in web development, data science, automation, and more.

C++

A powerful, high-performance language ideal for system/software development, game programming, and applications requiring real-time processing.

JavaScript

A dynamic language essential for web development, enabling interactive and dynamic content within browsers and on servers (Node.js).

Data Bases

SQL

A specialized language for managing and querying relational databases, critical for data retrieval and manipulation.

Mongo DB

A NoSQL database language using JSON-like documents, valued for flexibility and scalability in handling large datasets.

Marckup Language and StyleSheet

HTML

The standard markup language for creating web pages, defining the structure and content of websites.

CSS

A stylesheet language used to control the presentation and layout of web pages, enhancing HTML with visual design.
